# 1. Risk Assessment Lab

Identify assets, threats, and vulnerabilities, then calculate risk scores and create a risk register to prioritize security risks.

# 2. ISO 27001 Gap Analysis

Compare an organization’s security controls against ISO 27001 requirements to identify compliance gaps and suggest remediation actions.

# 3. NIST CSF Security Audit

Evaluate cybersecurity maturity using the NIST CSF framework (Identify, Protect, Detect, Respond, Recover) and provide a scoring report.

# 4. Vendor Risk Assessment

Assess third-party vendors based on security, compliance, and operational risk using a structured vendor risk scoring model.

# 5. Incident Response Plan

Develop a structured plan for detecting, responding to, and recovering from security incidents such as ransomware or data breaches.

# 6. Business Continuity & Disaster Recovery (BCDR)

Define recovery strategies, RTO/RPO, and procedures to ensure business operations continue during disasters or outages.

# 7. Security Awareness Program

Create policies and training materials to educate employees about phishing, password security, and safe cybersecurity practices.

# 8. Compliance Mapping (PCI DSS, HIPAA, GDPR)

Map security controls across multiple compliance frameworks to identify overlaps, gaps, and compliance readiness.
